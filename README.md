## Hexlet project 
### CLI App - Compares two configuration files and shows a difference.

---

## Setup
### Installing dependencies 

```bash
$ make install
```

### Run eslint 

```bash
$ make lint
```

### Run tests

```bash
$ make test 
```

### Installing gendiff app
#### in Linux

```bash
$ make gendiff-linux
```

#### in Windows

```bash
$ make gendiff-windows
```

### UnInstalling gendiff app 

```bash
$ make gendiff-uninstall
```
